## Semantic HTML

>Betekent dat je tags gebruikt met een betekenis, die iets zeggen over wat er binnen de tags te vinden is. (In plaats van alleen maar divs)

Voorbeelden:

```` 
<header> </header>
<main> </main>
<nav> </nav>
<footer> </footer>
<article></article>
<section></section>
<aside></aside>
````
